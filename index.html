This tool turns images into line screen halftone using WebGL shaders. The creation of this tool was largely inspired by the book cover of "Poor Charlie's Almanack", and motivation to learn to vibe code with Claude Code.

During the creation of this tool, I referenced heavily with the CMYK Halftone Emulator by Daniel Petho and also paper shaders by Paper. Thank you so much.

Error/Feedback: Please create an issue on Github repo.
